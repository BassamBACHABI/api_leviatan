# API LEVIATAN :

### Pour cloner le projet

```bash
git clone https://github.com/BassamBACHABI/api_leviatan.git
```

### Pour démarrer docker 

```bash
docker-compose build

docker-compose up -d
```

### Lien de la documentation de l'API (à executer dans un navigateur)
```bash
127.0.0.1:8000/api
```
