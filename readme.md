# API LEVIATAN :

### Pour cloner le projet

```bash
git clone git@github.com:yoanbernabeu/symfony6-php8-in-docker-compose.git
```

### Pour démarrer docker 

```bash
docker-compose build

docker-compose up -d
```

### Lien de la documentation de l'API (à executer dans un navigateur)
```bash
127.0.0.1:8000/api
```
